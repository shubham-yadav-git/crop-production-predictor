Technologies Used

This webapp is designed using html, css, bootstrap(few features) for frontend and connected to Machine Learning Model using Flask (Python framework).

We developed our model in three steps:

  1. Data Pre-processing
  2. Model Training
  3. Model Analysis

Data Pre-processing

Initially the data set with which we started was very big. The credit for data goes to data.gov.in We used two data set one included district vise data set of production of size(102800 X 5) and a rainfall data set. The both data sets of contains data of years from 1997-2015. After merging both data sets properly. We filtered most frequent crops in data sets. The description of the data set:
